# Material Color & Flat Color

> Reference from [flat design color] and [material design color]

## Usage
### Android
- copy color.xml to your project


### iOS:
- copy ios/color/
- MaterialColor.h
- MaterialColor.m
- FlatColor.h
- FlatColor.m
- In code:
```objc
//example
#import "MaterialColor.h"
self.textColor = [MaterialColor red];
```
